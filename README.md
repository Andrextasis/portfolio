# Andre Moro Portfolio

Static portfolio package for Andre Emanuel Moro Puma.

The current homepage direction is a compact dark personal portfolio inspired by tedawf.com: short intro, stacked visual, direct links, work/education tabs, featured projects, and notes for interviewers.

## Files

- `index.html` - main portfolio page.
- `styles.css` - responsive visual system.
- `script.js` - reader-path tabs.
- `assets/andre-moro-profile.png` - portfolio portrait.
- `docs/andre-moro-cv-behavioral-ux-research.html` - default CV.
- `docs/andre-moro-cv-people-org-psychology.html` - alternate CV.
- `docs/resume.css` - shared one-page resume template for web and A4 print.
- `docs/andre-moro-cv-behavioral-ux-research.pdf` - generated PDF version.
- `docs/andre-moro-cv-people-org-psychology.pdf` - generated PDF version.
- `docs/first-30-days-research-diagnostic.html` - research work sample.
- `docs/claim-support-matrix.html` - proof and privacy matrix.
- `docs/linkedin-online-audit.md` - LinkedIn and public presence audit.

Open `index.html` directly in a browser or deploy the whole folder to Vercel, Netlify, or GitHub Pages.
